HTML

  • Официальной спецификации HTML 1.0 не существует. До 1995 года существовало множество неофициальных стандартов HTML.
    Чтобы стандартная версия отличалась от них, ей сразу присвоили второй номер.
  • Четвёртая версия HTML (4.0) вышла 18 декабря 1997 года, а пятая (HTML5) только спустя 17 лет — 28 октября 2014 года.
  • HTML-теги могут быть как парными, например, <div></div>, так и непарными, например, <hr> или <br>.
  • HTML входит в тройку основных китов (HTML, CSS, JavaScript), с помощью которых строится веб-страница.
  • HTML-страница состоит из базовых HTML-тегов, которые применяются для создания большинства веб-страниц.
    Большая часть html-тегов при создании веб-страниц, как правило, не используется, а применяются только часто используемые и базовые.

CSS

  • Первая версия CSS была опубликована 17 декабря 1996 года.
  • До появления CSS веб-страницы оформлялись исключительно средствами HTML и непосредственно внутри содержимого документа.
    С приходом CSS появилась возможность отделять содержание и представление документа — код HTML хранился в одном файле, а стили CSS в другом.
  • Примечательный факт, что все веб-браузеры уже имеют свои встроенные стили отображения веб-страниц, но с помощью CSS их можно перезаписать "под себя".
  • На данный момент самой популярной и масштабно-разрабатываемой версией CSS является CSS3.
    От предыдущий версий эта отличается наличием по сути самостоятельно разрабатываемых модулей, таких как Flex, Grid, Animations, Transitions и тд.
  • C помощью CSS3 можно создавать полноценные анимационные элементы и даже целые композиции без использования JavaScript.
    Также внедрена поддержка градиентов, теней, сглаживания и прочее.

JavaScript

  • Первая версия языка JavaScript была разработана Бренданом Эйхом за 10 дней в 1995 году в компании Netscape.
  • Первоначально язык назывался Mocha, потом был переименован в LiveScript.
  • На JavaScript оказали влияние такие языки, как C и Java.
    Поскольку последний в то время был достаточно популярен, то LiveScript было решено переименовать в JavaScript.
  • Наиболее широкое применение JavaScript находит в браузерах (хотя может использоваться и не только в них) для придания веб-страницам интерактивности.
  • Языком JavaScript не владеет какая-либо компания или организация, а название «JavaScript» является зарегистрированным товарным знаком корпорации Oracle в США.
